# Session 49 Summary

**Date:** ~Oct 2025 (Arc 7)
**YouTube:** [Session 49](https://www.youtube.com/watch?v=vnvKznm_eWg)
**Arc:** Arc 7 — Rymes Return / Greenwoods / Grimordia

## Synopsis

**Rumpelstiltskin** (Memphis Cadwallader) appears during the Grimordia escape, invoking a fey contract binding Drizella to deliver four treasures. The party discovers the contract uses **false pretences** and burns it using the Spectral Key + Rumpelstiltskin's true name. He reveals his true form — a being of flame with contracts orbiting him — and demands the Ring + a mysterious book before departing. The party escapes via rope bridge as the castle collapses; Maleficent's breath attack fouls the river.

## Key Events

- Rumpelstiltskin invokes fey contract; party burns it using Spectral Key + true name
- Rumpelstiltskin's true form: being of flame with orbiting contracts
- He demands the Ring + a mysterious book before departing
- Castle collapse during escape; Maleficent's breath fouls the river

## Lore Reveals

- **Fey Contract Breaking** — Methods: (1) fey breaks it, (2) fey owed a debt, (3) killing the fey, (4) exit conditions. Burning + true name = breaking power.
- **Rumpelstiltskin's True Form** — Being of flame. Contracts orbit him. Major fey/devil collector.
